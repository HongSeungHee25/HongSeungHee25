## **🚀 About Me**
### 👋 안녕하세요! 끊임없이 성장하는 **백엔드 개발자 홍승희**입니다.  

저는 **트래블포트 API를 활용한 항공권 예약, 발권, 환불 시스템 개발** 및  
**외부 항공사 데이터를 실시간으로 수집하는 시스템 구축** 경험을 보유하고 있습니다.  

백엔드 개발자로서, **API 성능 최적화와 효율적인 데이터 처리**를 중요하게 생각하며,  
더 빠르고 안정적인 시스템을 만들기 위해 끊임없이 고민하고 있습니다.
